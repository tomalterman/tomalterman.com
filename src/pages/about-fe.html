Frontend Developement

A product can only be as great as the code it's written in. I have therefore always been involved in frontend development to ensure that the best possible experience gets delivered.

I have extensive knowledge of web technologies and have produced both interactive prototypes and large scale client-side interfaces for a range of organisations.

I have also lead, hired and trained frontend development teams.

  • Designing and implementing a scalable CSS framework using the principle of OOCSS
  • Producing fully accessible experiences using progressive enhancement & graceful degradation
  • Building client-side applications using JS frameworks
